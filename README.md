# Library Management System

## 📌 Description
This is a Library Management System developed using Java Swing and MySQL.

## 🛠 Technologies Used
- Java (Swing)
- MySQL
- NetBeans IDE

## ✨ Features
- User Login System
- Book Management
- Student Management
- Issue and Return Books
- Activity Log 

## ▶️ How to Run
1. Open project in NetBeans
2. Configure MySQL database
3. Run Login.java file
